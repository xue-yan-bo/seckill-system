# 🎨 前端项目（Frontend）

> Vue 3 + Vite + Element Plus

## 📂 这是前端代码目录

**技术栈**：
- Vue 3 - 前端框架
- Vite - 构建工具
- Element Plus - UI组件库
- Pinia - 状态管理
- Vue Router - 路由管理
- Axios - HTTP请求

## 🚀 启动方式

```bash
npm install    # 安装依赖（首次）
npm run dev    # 启动开发服务器
```

**访问地址**：http://localhost:3000

## 📁 目录说明

- `src/api/` - API接口定义
- `src/views/` - 页面组件
- `src/stores/` - 状态管理
- `src/router/` - 路由配置
- `src/utils/` - 工具函数

## 🎯 主要功能

- ✅ 用户登录注册
- ✅ 商品浏览搜索
- ✅ 秒杀活动参与
- ✅ 订单管理
- ✅ 个人中心

🔄 前端 vs 后端对比

🎨 前端特征（frontend/）

  • 语言：JavaScript
  • 框架：Vue 3
  • 文件：.vue .js .html
  • 配置：package.json
  • 工具：VSCode
  • 命令：npm install / npm run dev
  • 端口：3000
  • 职责：用户界面和交互

⚙️ 后端特征（backend/）

  • 语言：Java
  • 框架：Spring Boot
  • 文件：.java .xml .yml
  • 配置：pom.xml
  • 工具：IntelliJ IDEA
  • 命令：mvn spring-boot:run
  • 端口：8081-8084
  • 职责：业务逻辑和数据处理
